Python — Metodi delle Liste, Dizionari (uso e metodi) e Libreria random

Foglio A4 riassuntivo, pronto per la stampa • Sintassi Python 3.x

1) Liste — metodi principali

Inserimento / Modifica

# append: aggiunge in coda (O(1) ammort.)
nums = [1,2]; nums.append(3)   # [1,2,3]

# extend: concatena un iterabile
nums.extend([4,5])             # [1,2,3,4,5]

# insert: inserisce in posizione (shift a destra)
nums.insert(1, 99)             # [1,99,2,3,4,5]

# assegnazione slice: sostituzione in blocco
nums[2:4] = [7,8]              # [1,99,7,8,4,5]

Rimozione / Estrazione

# remove: rimuove la PRIMA occorrenza
nums.remove(99)

# pop: rimuove e ritorna (default: ultima)
x = nums.pop()      # toglie 5
y = nums.pop(0)     # toglie 1 dalla posizione 0

# clear: svuota la lista
nums.clear()

Ricerca / Conteggio

nums = [3,7,7,8,10]
# index: indice della prima occorrenza
i = nums.index(7)           # 1
# index(start, stop) opzionali
j = nums.index(7, 2)        # 2

# count: numero di occorrenze
c = nums.count(7)           # 2

Ordinamento / Vista

# sort: in-place (ordina la lista)
nums.sort()                 # crescente
nums.sort(reverse=True)     # decrescente

# key: funzione estrazione chiave
words = ["aa","b","ccc"]
words.sort(key=len)         # ["b","aa","ccc"]

# reverse: in-place, inverte ordine
nums.reverse()

# copy: copia superficiale (shallow)
p = nums.copy()

Operazioni utili

# slicing (copia porzione)
sub = nums[1:4]

# concatenazione
a = [1,2] + [3,4]           # [1,2,3,4]

# moltiplicazione (ripetizione)
zeros = [0] * 5             # [0,0,0,0,0]

# membership test
if 7 in nums: ...

# list comprehension (crea liste)
squares = [x*x for x in range(5)]
Complessità (indicativa)
  • append, pop() in coda: ≈ O(1) ammortizzato
  • insert(i,x), pop(i), remove(x): ≈ O(n)
  • sort: ≈ O(n log n)
  • Indicizzazione lst[i]: O(1)

2) Dizionari — quando usarli & metodi chiave

Quando usarli Usa un dizionario quando vuoi mappare una chiave a un valore (lookup veloce), ad esempio: punteggi per nome, inventari per ID, configurazioni per etichetta. A differenza delle liste, l’accesso è per chiave, non per posizione.

Creazione & Accesso

user = {"nome":"Matteo", "eta":18}
x = user["nome"]       # "Matteo" (KeyError se manca)
y = user.get("scuola", "N/D")  # default se assente

# Aggiunta / aggiornamento
user["eta"] = 19
user.update({"citta":"Torino"})

Rimozione & Default

# pop: rimuove e ritorna (KeyError se manca, salvo default)
eta = user.pop("eta", None)

# popitem: rimuove una coppia arbitraria (LIFO da 3.7+)
k, v = user.popitem()

# setdefault: ottieni chiave o iniziala
counts = {}
counts.setdefault("a", 0)
counts["a"] += 1

Viste & Iterazione

d = {"a":1, "b":2}
keys = d.keys()          # vista dinamica
vals = d.values()
items = d.items()

for k, v in d.items():
    print(k, v)

Copia, pulizia, costruzione

# copy: shallow copy
d2 = d.copy()

# clear: svuota
d.clear()

# fromkeys: crea con chiavi date
d3 = {}.fromkeys(["x","y"], 0)  # {"x":0,"y":0}

# dict comprehension
squares = {x: x*x for x in range(4)}
Scelte tra strutture
  • Lista: ordine posizionale, duplicati ammessi, ideale per sequenze.
  • Dizionario: accesso per chiave, lookup veloce, ideale per etichette → valori.
  • Set: collezione di elementi unici, test d’appartenenza veloce.
  • Tuple: sequenze immutabili, utili come chiavi di dict o per raggruppare dati fissi.

3) Libreria random — funzioni più usate

Valori numerici

import random as rnd

rnd.random()      # float [0.0, 1.0)
rnd.uniform(a,b)  # float in [a, b]
rnd.randint(a,b)  # intero in [a, b]
rnd.randrange(0, 10, 2)  # pari tra 0..8

Sequenze

items = ["a","b","c","d"]

rnd.choice(items)       # 1 elemento
rnd.choices(items, k=3) # con ripetizione
rnd.sample(items, 2)    # senza ripetizione
rnd.shuffle(items)      # in-place

Riproducibilità

rnd.seed(42)  # fissa la sequenza pseudo-casuale
# stesso seed ⇒ stessi risultati, utile per test

Distribuzioni (cenni)

rnd.gauss(mu=0, sigma=1)   # normale
rnd.expovariate(lambd=1)   # esponenziale
rnd.triangular(low, high, mode)

Consigli pratici

  • Usa shuffle per mescolare liste (es. mazzi).
  • choice per estrarre un elemento; sample per estrazioni senza ripetizioni.
  • Imposta seed nei test per risultati ripetibili.
  • Per sicurezza/criptografia usa secrets al posto di random.

Note veloci

  • Le liste sono mutabili: attenzione a passare riferimenti tra funzioni (usa lst.copy() se serve una copia).
  • I dizionari mantengono l’ordine di inserimento da Python 3.7+.
  • random è pseudo-casuale: non usarlo per password/chiavi segrete.
